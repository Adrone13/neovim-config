# Key mapping

```
Open explorer (netrw):                Space+p+v
Search in current file:               Space+f
Search in working directory:          Space+F
Search selected in current file:      Space+f (In visual mode)
Search selected in working directory: Space+F (In visual mode)
Search file by name:                  Space+p+f
Search in current git project:        Ctrl+p

Harpoon add file:                     Space+a
Harpoon toggle quick menu:            Ctrl+e
Harpoon open 1 file:                  Ctrl+h
Harpoon open 2 file:                  Ctrl+t
Harpoon open 3 file:                  Ctrl+n
Harpoon open 4 file:                  Ctrl+s

Git open UI:                          Space+g+s

Editor undo:                          u
Editor redo:                          Ctrl+r
Editor delete word:                   d+a+w
Editor delete word and go to insert:  c+a+w
Editor format indents in block:       =+a+p
Editor format indernts whole file:    gg+=+G

Editor split right:                   :vs
Editor split down:                    :sp
Editor select right split:            Ctrl+w+<Right> 
Editor select left split:             Ctrl+w+<Left>
Editor select top split:              Ctrl+w+<Up>
Editor select bottom split:           Ctrl+w+<Down>

Autocomplete confirm:                 Enter
Autocomplete show suggestions:        Option+c

LSP go to definition:                 g+d
LSP hover info:                       K
LSP rename:                           Space+v+r+n
LSP show code actions                 Space+v+c+a
```
